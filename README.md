## Setup

1.  Ensure Anaconda is installed locally (run conda --version to check)
2.  Install the following packages locally (run conda install or pip install):
    1.  numpy
    2.  pandas
    3.  matplotlib
    4.  seaborn
    5.  statsmodels
    6.  plotly
